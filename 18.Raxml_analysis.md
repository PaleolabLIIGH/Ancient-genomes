### 18.2 Get the script

Go to yoir work directory and copy the script:
```
cd /mnt/Cromosoma/mavila/lcarrillo/shotgun-pathogens/Mapping_hg38-concatenado/Unmapped/Salmonella/genotyping #Modify the path
cp /mnt/Cromosoma/mavila/backup/lcarrillo/scripts/raxml-ng.sh .
nano raxml-ng.sh
```

Modify the notification email, as well as the name of the fasta file you obtained with MAFFT. It is also good to verify that the model used is the one that has been used in previously published papers. 
This is the model used in the works with Salmonella enterica.
Next modify the resources, Raxml can give you the real resourses that you need with the following code:
```
module unload gcc/5.1.0
module load gcc/8.3.0
module load openmpi/gcc/64/1.8.5
module load raxml-ng/0.9.0
raxml-ng-mpi --parse --msa SalmonellasnpAlignment_95pdNo041No091noParaB_mafft.fas --model GTR+G --prefix test
```

In this case we have 237 sequences with 93,657 sites, 62,495 effective patterns, 1.88% gaps, 12.98% Invariant sites, 3.6 GB Estimated memory and 16 recommended threads, so in this case I modify the script as follow:
```
#$ -pe openmp 16        # 16 threads
#$ -l vf=5G             # 5 GB de RAM
```
and modify the threads in this part too:
```
raxml-ng-mpi --all --msa SalmonellasnpAlignment_95pdNo041No091noParaB_mafft.fas --model GTR+G --seed 12345 --threads 16 --bs-trees autoMRE
```
The script that you copied has a seed of 2, but is better to use a long seed as 12345, so do not forget to modified that too.

### 18.3 Run the script
```
qsub raxml-ng.sh
```
