Create a folder to perform the Kraken2 analysis and use a loop to create direct links of the .bz files to the new folder:

```
mkdir kraken2
for file in $(find /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped -type f -name "*bz2*"); do
```
