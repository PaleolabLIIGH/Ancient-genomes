Files with the extensions *k2.report and *.kraken2 will have been generated. To learn more about them, visit the page Kraken 2 Manual.

As you read on the previous page, the format of the reports needs to be changed for further analysis. So, create a new folder and process the *.report files:

```
mkdir k2_new_reports
for file in *.report; do
    # Create a new txt file with cat
    cat "$file" > "${file}.txt"

    # Move the txt file to the k2_new_reports folder
    mv "${file}.txt" k2_new_reports
done
```

Now go to the new folder and process the new txt files there:

```
cd k2_new_reports
for txt_file in *.txt; do
    # Use cut to extract specific columns
    cut -f1-3,6-8 "$txt_file" > "${txt_file%.txt}_std_report.txt"

    # Create the new file with the k2.std.report extension
    mv "${txt_file%.txt}_std_report.txt" "${txt_file%.txt}.kraken2.report"
done
```
