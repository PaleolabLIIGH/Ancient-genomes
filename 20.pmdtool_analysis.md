### 20.1 Study about the analysis
You can learn more about it here:
```
https://github.com/pontussk/PMDtools
https://www.pnas.org/doi/full/10.1073/pnas.1318934111
```
You can adopt the strategy that best suits your workflow. In my case, I prefer to have each PMD file (levels 0, 1, 2, and 3) within each subfolder for each individual. However, you can work with them 
in a separate folder or wherever you prefer. Here, I'll show you how to create them and place them inside each individual's subfolder.

### 20.2 Process the bam files for each PMD level
```
cd /mnt/Cromosoma/mavila/lcarrillo/HSJN-concatenado/Mapping_hg38/Mapping_mt/Schmutzi/
# Working directory
wdir="/mnt/Cromosoma/mavila/lcarrillo/HSJN-concatenado/Mapping_hg38/Mapping_mt/Schmutzi"

# Loop to process BAM files
for file in $(find $wdir/*H*/ -type f -name "*rCRS.MD.bam"); do
    module load htslib/1.2.1
    module load python26/2.6.9
    module load samtools/1.2
    
    # Process the file with pmdtools for each PMD level
    for level in 0 1 2 3; do
        samtools view -h -q 30 $file | python /mnt/Cromosoma/mavila/vvilla/scripts/pmdtools.py --header --threshold $level | samtools view -Sb - > "${file%.MD.bam}.pmd${level}.MD.bam"
        echo "Processed file: $file with PMD level $level"
    done
done
```
This loop generates new BAM files for each PMD level.

### 20.3 Index the processed BAM files
```
# Loop to index the processed BAM files
for file in $(find $wdir -type f -name "*pmd*bam"); do
    module unload samtools/1.2
    module load samtools/1.16.1
    samtools index $file
done
```

### 20.4 Organizing files
# Loop to move the files to their respective PMD folders inside each individual subfolder
```
for subfolder in $(find $wdir/*H*/ -type d); do
    for level in 0 1 2 3; do
        mkdir -p "$subfolder/pmd$level"
        mv "$subfolder"/*pmd${level}* "$subfolder/pmd$level/"
    done
done
```
