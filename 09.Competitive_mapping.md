### 9.1 Load the reference genomes
The first thing you need to do is create a folder with the reference genomes of the genus you are interested in:

```
cd /mnt/Cromosoma/mavila/lcarrillo/ #Modify path
mkdir genomes_bacteria
cd genomes_bacteria
```

Now go to the NCBI website and search for the reference genomes of the genus you are interested in and download the fasta files. Rename them for easier identification:

```
ls Salmonella_enterica_*
```
```
#Output:
Salmonella_enterica_Enteritidis.fna
Salmonella_enterica_Heidelberg.fna
Salmonella_enterica_Newport.fna
Salmonella_enterica_P_A_ATCC.fna
Salmonella_enterica_Paratyphi_A.fna
Salmonella_enterica_Paratyphi_B.fna
Salmonella_enterica_Paratyphi_C.fna
Salmonella_enterica_Typhi_CT18.fna
Salmonella_enterica_Typhi_Ty2.fna
```

Concatenate all the fasta files into a single fasta file:

```
cat Salmonella_enterica_Enteritidis.fna > salmonella_competitivo.fna
cat Salmonella_enterica_Heidelberg.fna >> salmonella_competitivo.fna
cat Salmonella_enterica_Newport.fna >> salmonella_competitivo.fna
```

Verify that all genomes were included:
```
grep ">" salmonella_competitivo.fna
```


Once you are notified by email that the job is completed, return to the competitive mapping folder and activate the necessary modules:

```
cd /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped/salmonella_competitiva #Modify path
module load htslib/1.2.1

module load samtools/1.16.1
```

Since you already know the accession numbers of each genome, which you observed with grep in a previous step, you can use these complete numbers or patterns to calculate the mapping by species with the 
following code:

```
# Pattern for Typhi str. CT18
for file in $(find /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped/salmonella_competitiva -type f -name "*uniq.bam*"); do #Modify path
  name=$(basename "$file" | cut -d "_" -f 1) 
  reads=$(samtools view $file | grep "AL51338" | wc -l) #Modify pattern
  echo $name $reads
done

# Pattern for Typhi Ty2
for file in $(find /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped/salmonella_competitiva -type f -name "*uniq.bam*"); do 
  name=$(basename "$file" | cut -d "_" -f 1) 
  reads=$(samtools view $file | grep "AE014613.1" | wc -l)
  echo $name $reads
done

# Pattern for Typhimurium str. LT2
for file in $(find /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped/salmonella_competitiva -type f -name "*uniq.bam*"); do 
  name=$(basename "$file" | cut -d "_" -f 1) 
  reads=$(samtools view $file | grep "NC_003" | wc -l)
  echo $name $reads
done
```

Once you have determined the correct species from the competitive mapping results, go back to the genomes folder and index that genome the same way you did with the competitive fasta file.

Return to your Unmapped folder and create a folder for mapping that species:

```
cd /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped #Modify path
mkdir Salmonella_enterica_Pc #Modify name
```

Copy the sequences to the folder and perform the mapping for that species just as you did in the previous steps.

Repeat the same steps for the other genera of your interest.
