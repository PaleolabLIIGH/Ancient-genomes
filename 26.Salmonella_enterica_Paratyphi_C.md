```
#doi
10.1016/j.cub.2022.06.094

To learn more about these analysis you can read files 13 and 14 prevously described.

grep "new_name=" /mnt/Cromosoma/mavila/lcarrillo/scripts/generation_of_ancient-salmonella-genomes_vcf_files.sh
```
