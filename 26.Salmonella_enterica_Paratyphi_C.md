Copy the script and modify `DIR` and `new_name`
```

grep "DIR=" /mnt/Cromosoma/mavila/lcarrillo/scripts/generation_of_ancient-salmonella-genomes_vcf_files.sh
grep "new_name=" /mnt/Cromosoma/mavila/lcarrillo/scripts/generation_of_ancient-salmonella-genomes_vcf_files.sh
```
```
#Output:
REF="/mnt/Cromosoma/mavila/lcarrillo/genomes_bacteria/ParatyphiC_NC_0121251.fasta" #Modify path
DIR="/panfs/pan5/mavila/lcarrillo/Salmonella-definitivo/other-ancient-samples" #modify path

DO NOT modify `REF` because the dictionary is already in that path.

Run the script
```
