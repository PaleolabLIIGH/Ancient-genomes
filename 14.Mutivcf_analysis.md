your samples and those from other studies). You need to trim the ends of your reads in the BAM files to mitigate the impact of DNA degradation on the 
phylogenetic analysis. To start, locate the directory containing your ancient genome BAM files.

Activate the following module and trim the first and last two bases of each read.
```

    echo "Indexing: $i"
    samtools index $i
done


# Iterate over each uniq.bam file in the directory
for BAM in $DIR/*uniq.bam; do
    # Extract the base name without the extension for output file naming
    BASENAME=$(basename $BAM .bam)

    # Step 3: Create realignment targets
    GenomeAnalysisTK -T RealignerTargetCreator -R $REF -I $BAM -nt 4 -o ${BAM}.MarkD.intervals

    # Step 4: Realignment around indels
    GenomeAnalysisTK -T IndelRealigner -R $REF -I $BAM -targetIntervals ${BAM}.MarkD.intervals -o ${BAM}.real.bam

    # Step 5: Call variants
    GenomeAnalysisTK -T UnifiedGenotyper -R $REF -I ${BAM}.real.bam -o ${BAM}.real.bam.vcf -nt 4 -stand_call_conf 30 --sample_ploidy 2 -dcov 250 --output_mode EMIT_ALL_SITES
done

#Create a directory only for vcf files
mkdir genotyping
mv *vcf* genotyping

#Rename the vcf files
cd genotyping
for file in *.vcf*; do
    new_name=$(echo "$file" | sed 's/_1-2.fastq.collapsed.unmapped.fastq.bwa.sort.rmdup.uniq.bam.real.bam//') #Modify the extension if needed
    mv "$file" "$new_name"
done

# Iterate over all .vcf files in the folder
for file in *.vcf; do
  # Get the base name without the extension
  base_name="${file%.vcf}"
  
  # Create a folder with the base name
  mkdir -p "$base_name"
  
  # Move the .vcf file and its corresponding .vcf.idx file to the folder
  mv "$base_name.vcf" "$base_name/"
  mv "$base_name.vcf.idx" "$base_name/"
done

exit
```
