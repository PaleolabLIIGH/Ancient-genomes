### 14.2 Get the uniq.bam file of your reference genomes
The `uniq.bam` files obtained by mapping your sequences against a reference genome should be compared against that reference genome in your phylogeny. To do this, you need to create a uniq.bam file 
for that reference genome, as well as for other genomes you want to include in the phylogeny, such as the outgroup, among others.

To achieve this, go to your directory with all your reference genomes that you will include in your phylogeny:

### 14.4 Obtaining VCF files
Up to this point, you have downloaded the fastq files for all the modern and ancient lineages that you will use in the phylogeny, organized into paired-end and single-end folders. You have removed 
the adapters, separated the human fraction, and mapped the non-human fraction against your reference genome, obtaining the uniq.bam files.

Now, we will obtain the VCF files for all of them. I will show how to do this in a single folder (in this case, `/mnt/Cromosoma/mavila/lcarrillo/mtb_dataset/animal-single/Mapping_hg38/Unmapped/mtb_ancestral`), 
but make sure to do this in all the folders of your dataset (in my case, all those in `/mnt/Cromosoma/mavila/lcarrillo/mtb_dataset/`).

```
#Written by Miriam Bravo
#Modified by Victoria Pastor and Laura Carrillo to work in loops:
cd /mnt/Cromosoma/mavila/lcarrillo/mtb_dataset/animal-single/Mapping_hg38/Unmapped/mtb_ancestral
module load htslib/1.2.1
module load gcc/5.1.0
module load bwa/0.7.13
module load samtools/1.16.1

#Indexing bam files:
for i in *.uniq.bam; do
    echo "Indexing: $i"
    samtools index $i
done

qlogin -pe openmp 12
cd /mnt/Cromosoma/mavila/lcarrillo/mtb_dataset/animal-single/Mapping_hg38/Unmapped/mtb_ancestral #Modify path
module load gatk/3.5
module load picard/2.6.0
# Define the reference genome paths for easier readability and modification
REF="/mnt/Cromosoma/mavila/lcarrillo/genomes_bacteria/MTB_ancestor.fasta" #Modify path
# Directory where the BAM files are located
DIR="/mnt/Cromosoma/mavila/lcarrillo/mtb_dataset/animal-single/Mapping_hg38/Unmapped/mtb_ancestral" #modify path
# Iterate over each uniq.bam file in the directory
for BAM in $DIR/*uniq.bam; do
    # Extract the base name without the extension for output file naming
    BASENAME=$(basename $BAM .bam)

    # Step 3: Create realignment targets
    GenomeAnalysisTK -T RealignerTargetCreator -R $REF -I $BAM -nt 4 -o ${BAM}.MarkD.intervals

    # Step 4: Realignment around indels
    GenomeAnalysisTK -T IndelRealigner -R $REF -I $BAM -targetIntervals ${BAM}.MarkD.intervals -o ${BAM}.real.bam

    # Step 5: Call variants
    GenomeAnalysisTK -T UnifiedGenotyper -R $REF -I ${BAM}.real.bam -o ${BAM}.real.bam.vcf -nt 4 -stand_call_conf 30 --sample_ploidy 2 -dcov 250 --output_mode EMIT_ALL_SITES
done
ls *vcf
```
