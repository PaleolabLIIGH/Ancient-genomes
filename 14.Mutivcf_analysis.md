### 14.5 Organize the vcf files
We should organize the files into one folder per individual. Therefore, we will use the following loop to achieve it.
You can create the name of the directories with the method of your preference. Here I show you two ways to do it!
#### Method 1: With grep
```
for file in *.vcf*; do
    dir_name=$(echo "$file" | grep -oP '^[^-]+-[^-]+'); #Modify regex if needed
    echo $dir_name
done
```

```
for file in *.vcf*; do
    dir_name=$(echo "$file" | grep -oP '^[^-]+-[^-]+');
    mkdir -p "$dir_name"
    mv "$file" "$dir_name/"
```
