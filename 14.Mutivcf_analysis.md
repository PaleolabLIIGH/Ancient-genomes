### 14.6 Obtaining VCF files of modern genomes
Up to this point, you have downloaded the fastq files for all the modern and ancient lineages that you will use in the phylogeny, organized into 
paired-end and single-end folders. You have removed the adapters, separated the human fraction, and mapped the non-human fraction against your 
reference genome, obtaining the uniq.bam files.

Now, we will obtain the VCF files for all of them. 
This is the script to study:
```
#!/bin/bash
#SBATCH --job-name=vcf_generation
#SBATCH --output=vcf_generation_%j.out
#SBATCH --error=vcf_generation_%j.err
#SBATCH --mail-type=END,FAIL
#SBATCH --mail-user=laura-carrillo-olivas@hotmail.com
#SBATCH --cpus-per-task=10
#SBATCH --mem=5G
#SBATCH --ntasks=1

### Written by Miriam Bravo
### Modified by Victoria Pastor to work in loop
### Modified by Laura Carrillo to work in fenix
### Creation of VCF files for .uniq.bam files only

# Load environment modules

module load picard/2.6.0
module load htslib/1.2.1
module load gcc/5.1.0
module load bwa/0.7.13
module load samtools/1.16.1
module load oracle-java/8u371
module unload gatk
module load gatk/3.5

# ===============================
# Define paths
# ===============================
REF="/mnt/data/mavila/lcarrillo/genomes_bacteria/Streptococcus_pneumoniae_GPSC83_A66.fasta"
DIR="$PWD"

# Create output directory if not exists
mkdir -p genotyping

# ===============================
# Index BAM files
# ===============================
for i in ${DIR}/*.uniq.bam; do
    echo "Indexing: $i"
    samtools index "$i"
done

# ===============================
# Generate VCFs
# ===============================
for BAM in ${DIR}/*uniq.bam; do
    RAW=$(basename "$BAM" .bam)
    SAMPLE="${RAW%%_1-2*}"  # Modify pattern if needed


    # Step 1: Create realignment targets
    java -jar $GATK_JAR -T RealignerTargetCreator -R "$REF" -I "$BAM" -nt 4 -o "${SAMPLE}.MarkD.intervals"

    # Step 2: Realignment around indels
    java -jar $GATK_JAR -T IndelRealigner -R "$REF" -I "$BAM" -targetIntervals "${SAMPLE}.MarkD.intervals" -o "${SAMPLE}.real.bam"

    # Step 3: Variant calling
    java -jar $GATK_JAR -T UnifiedGenotyper -R "$REF" -I "${SAMPLE}.real.bam" -o "${SAMPLE}.vcf" -nt 4 -stand_call_conf 30 --sample_ploidy 2 -dcov 250 --output_mode EMIT_ALL_SITES

    # Create directory for each sample and move the VCFs
    mkdir -p "genotyping/${SAMPLE}"
    mv "${SAMPLE}.vcf" "genotyping/${SAMPLE}/"
    mv "${SAMPLE}.vcf.idx" "genotyping/${SAMPLE}/" 2>/dev/null
done

```
