### 12.1 Prepare your files
Convert aligned reads from BAM files to into a pileup format, which shows the base calls at each position in the genome. This format includes
information such as the coverage depth and the base call from each read.

```
cd /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped/salmonella_concatenado
module load htslib/1.2.1
module load gcc/5.1.0
module load bwa/0.7.13
module load samtools/1.16.1
for file in $(find /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped/salmonella_concatenado -type f -name "*uniq.bam"); do
  samtools mpileup -f /mnt/Cromosoma/mavila/lcarrillo/genomes_bacteria/Salmonella_enterica_Paratyphi_C.fna -C 50 $file -o $file.pileup
done
ls *bam.pileup
```

Using the previous output with the Python script that calculates coverage by windows. In that script, you will find this part: `# length of slice to 
average over L = 100`. Edit the window size if you wish; this means it will calculate the coverage for every 100 base pairs (bp).

If you have only one chromosome in your genome use the following code:
```
module load python35/3.5.2
for file in $(find /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped/salmonella_concatenado -type f -name "*bam.pileup"); do
  python /mnt/Cromosoma/mavila/mbravo/5.Utilities/Circos/circos-0.69-6/average-pileup.py $file > ${file%.pileup}_coverage100pb.txt
done
```

 But this is not the case, for example Salmonella has a chromosome and a plasmid, so instead of the previous script use this code:
```
cp /mnt/Cromosoma/mavila/backup/lcarrillo/scripts/average-pyleup_modified.py /mnt/Cromosoma/mavila/lcarrillo/scripts/

  python /mnt/Cromosoma/mavila/lcarrillo/scripts/average-pyleup_modified.py $file > ${file%.pileup}_coverage100pb.txt

```
