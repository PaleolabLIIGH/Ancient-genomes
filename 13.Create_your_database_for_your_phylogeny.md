To build a phylogenetic tree, you need a reference dataset. In all articles on ancient or modern pathogens where a phylogeny has been constructed, they mention which dataset was used, and you should use
the same one. If there is no dataset and you are going to build one from scratch, make sure to download genomes from different parts to have a diverse dataset.

The first step is to download the FASTQ files using the accession numbers usually provided in the "Data Available" section of the article where the phylogeny was constructed. Some are from NCBI, and 
others are from ENA Browser.

If the targeted sequences are from ENA please use the Method 1
If the targeted sequences are from NCBI pleae use the Method 2

### 13.6 Rename your samples
When all the genomes have finished downloading your need to add the name of the sample folder to the names of each of the fastq files, this will make your life a lot easier in downstream analyses. 
Before you use this make sure to test it using the echo command:
```
cd ancient-paired
for d in *; do
  [ -d "$d" ] && cd "$d" || continue
  for f in ./*.gz; do
    echo mv -v "${f}" "${d}_${f#./}"
  done
  cd -
done
```

### 13.8 Modify the File Name
As you may recall, for many of the scripts we use, it's preferable that the sample name is separated by `-` while the rest is separated by `_`. In these examples, we see that the name consists of 
the name followed by the accession number, then "R" and finally the extension .fastq.bz2. However, some names have `-` and others have `_`, and we need to standardize them.
For example, `*U*` samples have the next pattern: `*U*_S*_*.fastq.bz2`, so we will use the next code for them:
```
for file in *U*_S*_*.fastq.bz2; do
    new_file=$(echo "$file" | sed 's/_/-/');
    echo "$new_file";
done
```

If the names are correct then use the next code:
```
for file in *U*_S*_*.fastq.bz2; do
    new_file=$(echo "$file" | sed 's/_/-/');
    mv "$file" "$new_file";
done
ls
```

```
#Output:
281U-2-SRR16922678_1.fastq.bz2    82U-SRR16922691_1.fastq.bz2         body_28-3_ERR650983_2.fastq.bz2
281U-2-SRR16922678_2.fastq.bz2    82U-SRR16922691_2.fastq.bz2         body_28-3_ERR650983_1.fastq.bz2
```

body samples have the next pattern: `body_*_*_*.fastq.bz2`, so we will use the next code for them:
```
for file in body_*_*_*.fastq.bz2; do
    new_file=$(echo "$file" | sed 's/_/-/; s/_/-/');
    echo "$new_file";
done
```
If they are correct then:
```
for file in body_*_*_*.fastq.bz2; do

ls
```
