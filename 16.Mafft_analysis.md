### 16.2 Modify the script
Go to your work directory
```
cd /mnt/Cromosoma/mavila/lcarrillo/shotgun-pathogens/Mapping_hg38-concatenado/Unmapped/Salmonella/genotyping
```

Rename the file with the extension *fasta
```
mv SalmonellashotgunsnpAlignment_pd95.fas SalmonellashotgunsnpAlignment_pd95.fasta #Modify the names
```

Get the script
```
cp /mnt/Cromosoma/mavila/backup/lcarrillo/scripts/mafft.sh .
```

Modify the name of the file
```
nano mafft.sh
#In the section input file
input_file="SalmonellashotgunsnpAlignment_pd95.fasta"
```

Run the script
```
qsub mafft.sh
```
