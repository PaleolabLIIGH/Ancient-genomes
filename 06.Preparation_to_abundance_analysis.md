### 6.1 Transform the reports to a matrix
Each Kraken report provides useful information for each individual. A quick way to visualize abundances is through Pavian, as we discussed earlier. However, to better filter these abundances, we need to 
transform these *report formats into a matrix format. One of the most commonly used formats is BIOM, which you can learn more about https://biom-format.org/

To do this, follow these steps:

```
qlogin
cd /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped/kraken2/k2_new_reports/
module load kraken-biom/1.0.1
for f in $(find /mnt/Cromosoma/mavila/lcarrillo/capturas_salmonella_myco/Mapping_hg38/Unmapped/kraken2/k2_new_reports/ -type f -name "*2.report"); do
    kraken_biom.py $f -o ${f%.kraken.report}.biom --fmt json
done
```

Now you have BIOM format files for each individual:

```
ls *biom
```

To create the matrix, use the following code. Ensure there are no spaces between each comma and sequence:

```
merge_otu_tables.py -i 021-H238-WISC_L001_R1-2_001.fastq.collapsed.unmapped.fastq.k2.report.kraken2.report.biom,041-H302-WISC_L001_R1-2_001.fastq.collapsed.unmapped.fastq.k2.report.kraken2.report.biom,091-H337bis-WISC_L001_R1-2_001.fastq.collapsed.unmapped.fastq.k2.report.kraken2.report.biom,099-H402-WISC_L001_R1-2_001.fastq.collapsed.unmapped.fastq.k2.report.kraken2.report.biom,109-H402c-WISC_L001_R1-2_001.fastq.collapsed.unmapped.fastq.k2.report.kraken2.report.biom -o merged_capturas_salmonella_myco.biom
```

Convert the BIOM matrix to TXT format using the following code:

```
biom convert -i merged_capturas_salmonella_myco.biom -o merged_capturas_salmonella_myco.biom.txt --to-tsv --header-key taxonomy
exit
```
